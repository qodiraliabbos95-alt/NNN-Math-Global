# 🧮 NNN-Math-Global

**ENGINE 1000+ MEGA UNIVERSAL MATH AI**

## 📋 Loyiha haqida

Bu loyiha universal matematik yechimlar tizimi bo'lib, quyidagi imkoniyatlarga ega:

- ✅ Tenglamalarni yechish (step-by-step)
- ✅ Hosila (derivative) hisoblash
- ✅ Integral hisoblash
- ✅ Formulalar bazasi
- ✅ Web interfeys (Streamlit)

## 📁 Fayllar

- `main.py` - Asosiy dastur kodi
- `app.py` - Streamlit web interfeys
- `requirements.txt` - Kerakli kutubxonalar

## 🚀 Ishlatish

### 1. Kutubxonalarni o'rnatish:
```bash
pip install -r requirements.txt
```

### 2. Terminal orqali ishlatish:
```bash
python main.py
```

### 3. Web interfeysni ishga tushirish:
```bash
streamlit run app.py
```
